# 🎓 Video Learning Platform (Angular)

A simple subject-based video learning web application built using **Angular**, where users can log in, select a subject, and watch related video content.

---

## 🚀 Features

* 🔐 User Authentication (Login)
* 📚 Subject Management (Create, Read, Update, Delete)
* 🎥 Video Management per Subject (Create, Read, Update, Delete)
* 🎥 Video Listing per Subject 
* ▶️ Video Streaming (Supabase Storage Integration)
* 💡 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** Angular
* **Backend/Auth:** Supabase
* **Storage:** Supabase (for video hosting)

---

## 📂 Project Structure (Overview)

* `auth` → Login functionality
* `subjects` → Subject selection UI
* `videos` → Video listing and playback
* `services` → API & Supabase integration

---

## ⚙️ Setup Instructions

1. Clone the repo:

```bash
git clone https://github.com/tayyab-balti/video-learning-platform-angular.git
```

2. Install dependencies:

```bash
npm install
```

3. Add environment variables:

* Supabase URL
* Supabase API Key

4. Run the app:

```bash
ng serve
```
